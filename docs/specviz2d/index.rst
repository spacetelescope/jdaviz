Specviz2d is a tool for visualization and quick-look analysis of 2D astronomical spectra.
It incorporates visualization tools with analysis capabilities,
such as Astropy regions and `specreduce <https://specreduce.readthedocs.io>`_
and :ref:`specutils` packages. In particular, it features interactive spectral
extraction of 2D spectra.
