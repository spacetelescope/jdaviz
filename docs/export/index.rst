Overview
========

Jdaviz allows you to export various data products created during your analysis:

**Data Products**
  - **Viewers**: Export viewer plots as images
  - **Datasets**: Export loaded and derived datasets
  - **Subsets**: Export spatial and spectral subsets/regions
  - **Plugin Tables**: Export tabular results from plugins
  - **Plugin Plots**: Export plots generated by plugins

Export Methods
==============

Data can be exported through the UI or programmatically:

**Via UI**: Most plugins have export buttons that save results to files.

**Via API**: Use the programmatic interface to export data:

.. code-block:: python

    # Export a dataset
    spectrum = specviz.get_data('my_spectrum')
    spectrum.write('exported_spectrum.fits')

    # Export a subset
    subset = specviz.get_subset('Subset 1')
    subset.write('subset.reg')
