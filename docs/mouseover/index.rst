Overview
========

Mouseover functionality provides real-time information about data as you move your cursor across viewers:

* **Image Viewer**: Display pixel coordinates, values, and WCS information
* **Table Viewer**: Show row data and column values under the cursor
* **Spectrum Viewer**: Display wavelength and flux values at cursor position
* **Profile Viewer**: Show profile values along the cursor position
